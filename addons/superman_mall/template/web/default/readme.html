以下是隔窗听雨对产品的一些想法和思路：

首先感谢大家对超人团队的信任与支持，让我们超级商城这么强大的模块在短时间内众筹成功。

其实我们诞生这个超级商城的想法很久了，开发这个商城的目的，就是为了让商户通过整合的营销工具（如：秒杀，抢购，拍卖，抽奖，砸金蛋等）把商品销售出去。以服务本地商家为结果的本地多商家商城。


为什么叫超级商城？

本商城是一个多商户入驻式的综合商城，利用线上与线下的关系，结合细腻的产品分类，营销功能，三级分销，PC端页面，以及后期增加活动专题页，比如：年货清仓专题页，情人礼物区，圣诞礼物抢购，中秋秒杀专区，最大程度的刺激消费者买单，让商家赚钱，平台自然会赚钱。


平台赢利点分析

  1. 商户入驻费用
  2. 广告费用
  3. 商品推广费用
  4. 营销工具收费（秒杀，抢购等等）
  5. 销售提成（美团等均价在10个点左右）
